# 90坦克大战 Web 开发记录（v5）

日期：2026-03-31

## 本轮目标

- 让 Tank90 的“可玩性/可测性”进入可迭代状态：敌人队列更贴近经典分型、玩家生命与重生完整闭环、道具系统可生成/可拾取/有效果/可回退。
- 解决操作手感问题：坦克贴边/贴坦克时的体积感与炮管溢出问题。
- 补齐端到端测试：无需依赖 Debug 面板，测试可通过受控注入稳定覆盖关键逻辑。
- 主基地（老鹰）替换为更还原的像素鹰形（按截图参考）。

## 本轮结论（代码层）

### 1) 敌方 archetype 对齐为 Basic/Fast/Power/Armor，并更新每关队列

- `EnemyArchetypeId` 从 `grunt/raider/heavy/sniper` 调整为 `basic/fast/power/armor`。
- `ENEMY_ARCHETYPES` 参数重新标定（速度/装填/弹速/转向频率/开火概率/追击倾向）。
- `levels.ts` 中 Stage 1–10 的 `enemyQueue` 同步改为新 ID，保证渲染与生成一致。

相关文件：

- `src/tank90/levels.ts`
- `src/tank90/core/update.ts`（spawn 时默认 archetype 改为 `basic`）

### 2) 生命系统：初始 4 命（3 reserve + 1 当前），死亡扣命并重生

- 新增 `playerLivesReserve`（初始 3），死亡时若 reserve > 0 则扣 1 并按关卡默认出生点重生；否则判定 LOST。
- 重生时统一重置：位置/朝向/开火冷却/弹速基线/出生护盾计时/道具层级归零。

相关文件：

### 3) 道具系统（6 种）+ 计时效果 + 受控生成

新增 `PowerUpKind`：`grenade/helmet/shovel/star/tank/timer`

- **生成**：每关独立计时器（约 7–10s 抖动），从 4×4 的候选点中择优；存在时 6.5–9s 会消失。
- **拾取**：与玩家 hitbox 发生 overlap 即生效。
- **效果**：
  - `grenade`：清屏（清除所有存活敌人）
  - `helmet`：护盾（延长无敌）
  - `shovel`：基地周围三面砖临时升级为钢，并在到期后恢复
  - `star`：火力升级（tier 0→3：弹速/装填提升；tier≥2 允许双发；tier≥3 可破钢）
  - `tank`：+1 reserve 命
  - `timer`：冻结敌人 AI/移动一段时间
- **提示文案**：拾取提示仅显示效果，不显示 kind（例如 `CLEAR ENEMIES`、`FREEZE ENEMIES` 等）。

相关文件：

- `src/tank90/core/types.ts`
- `src/tank90/core/state.ts`
- `src/tank90/core/update.ts`
- `src/tank90/core/render.ts`

### 4) 碰撞手感与造型：坦克碰撞使用更小 hitbox；炮管不再溢出坦克体

- 碰撞改为 “车身略小 hitbox”：`TANK_HITBOX_INSET = 1`，减少斜角卡顿与贴边误碰撞感。
- 坦克炮管绘制改为**完全在 16×16 内**（向内收 1px），避免视觉上“炮管也有体积”。

相关文件：

- `src/tank90/core/update.ts`
- `src/tank90/core/render.ts`
- `src/tank90/TankBattle90View.tsx`（命数小图标炮管同样收进框内）

### 5) 渲染改动：道具置顶、草覆盖、Armor 视觉增强、主基地像素鹰

- **渲染层级**：
  - 地形 pass1（砖/钢/水/冰）→ 基地 → 坦克/子弹 → 草 pass2（覆盖）→ 道具（最高层）
- **Armor 敌人**：
  - 移除 “hp>1 的额外血条/点” 类视觉，改为外框强调（更接近经典装甲感）。
- **基地（老鹰）**：
  - 使用参考截图提取的 32×32 位图绘制（黑底 + 灰鹰 + 仅头部红点）。
  - 增加“洞填充”以确保鹰身不出现镂空（内部封闭的 `.` 自动填灰）。

相关文件：

- `src/tank90/core/render.ts`

### 6) e2e：新增 `?e2e=1` 下的受控注入接口，测试不再依赖 Debug 面板

在 `TankBattle90View.tsx` 中仅当 `?e2e=1` 时注入：

- `window.__tank90_e2e.getSnapshot()`
- `injectEnemyBulletOnPlayer()`
- `placePowerUpOnPlayer(kind)`
- `setPowerTier(tier)`
- `killAllEnemiesAndWin()`

Playwright 用这些接口稳定覆盖：扣命重生/道具效果/第 10 关结算等。

相关文件：

- `src/tank90/TankBattle90View.tsx`
- `e2e/tank90.spec.ts`
